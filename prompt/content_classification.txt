内容分类任务：请分析以下文本的内容类别，并从预定义类别中选择最合适的一个。

文本内容：
{text}

预定义类别：
- financial: 金融财经内容，包含股票、证券、银行、投资、理财等
- tech: 科技内容，包含人工智能、编程、软件开发、互联网技术、电子产品等
- news: 新闻内容，包含时事新闻、社会新闻、政治新闻等
- education: 教育内容，包含学习资料、教育资讯、培训课程等
- medical: 医疗内容，包含疾病、药品、治疗方法、健康资讯等
- legal: 法律内容，包含法律法规、案例分析、法律服务等
- other: 其他不属于以上类别的内容

分类要求：
1. 只能选择一个最贴切的类别
2. 提供分类置信度（0.0-1.0）
3. 简要说明分类理由
4. 判断内容是否适合进行知识图谱构建

请以JSON格式返回结果，包含以下字段：
- category: 选择的类别（字符串）
- confidence: 置信度（0.0-1.0）
- reasoning: 分类理由（不超过100字）
- supported: 是否支持知识图谱构建（布尔值）
- is_financial_content: 是否为金融内容（布尔值）

示例输出：
```json
{
  "category": "tech",
  "confidence": 0.85,
  "reasoning": "文本主要讨论人工智能技术发展",
  "supported": true,
  "is_financial_content": false
}
```
