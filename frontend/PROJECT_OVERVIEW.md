# 知识图谱可视化平台 - 项目概览

## 🎯 项目简介

基于知识图谱API路由能力构建的轻量级展示页面，采用大厂设计规范，提供现代化的用户界面和流畅的用户体验。

## 🏗️ 系统架构

### 前端技术栈
- **HTML5 + CSS3 + JavaScript (ES6+)**
- **响应式设计** - 完美适配桌面端、平板和移动设备
- **现代化UI组件** - 遵循大厂设计规范
- **网络图可视化** - 集成vis-network库实现关系网络展示
- **模块化架构** - 高度可维护和可扩展

### 后端技术栈
- **FastAPI** - 高性能异步Web框架
- **SQLAlchemy** - 异步ORM框架
- **ChromaDB** - 向量数据库存储
- **大模型集成** - 实体识别和关系抽取

## 🚀 核心功能

### 1. 内容处理模块
- **智能文本分析** - 自动识别文本中的实体和关系
- **内容分类** - 自动分类内容类型（科技、金融、医疗等）
- **实时处理** - 异步处理，实时反馈处理进度
- **结果展示** - 直观展示提取的实体和关系

### 2. 实体管理模块
- **实体列表** - 分页展示所有实体
- **搜索过滤** - 支持按名称、类型搜索和过滤
- **实体详情** - 查看实体的完整信息和关联关系
- **数据可视化** - 卡片式布局，信息层次清晰

### 3. 网络分析模块
- **关系网络图** - 可视化展示实体间的关系
- **交互式探索** - 支持节点拖拽、缩放、点击等交互
- **多深度分析** - 支持1-3度关系网络分析
- **动态布局** - 自动优化网络布局算法

## 🌐 部署说明

### 开发环境
```bash
# 启动前端服务
cd frontend
python3 -m http.server 8081

# 启动后端服务（在另一个终端）
cd ..
uvicorn app.main:app --host 0.0.0.0 --port 8001 --reload
```

### 访问地址
- **主系统**: http://localhost:8081/index.html
- **演示页面**: http://localhost:8081/demo.html
- **测试页面**: http://localhost:8081/test.html
- **API文档**: http://localhost:8001/docs
